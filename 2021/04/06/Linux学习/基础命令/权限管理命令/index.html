权限命令chmod

命令名称：chmod
英文：change the permissions mode of a file
命令所在路径：/bin/chmod
执行权限:所有用户
语法：chmod [{ugoa} {+-=} {rwx}][文件或目录]
[mode=421][文件或目录]
-R 递归修改 当操作目录时会将目录下的所有文件都改为一个修改值
功能描述：改变文件或目录权限
所有者u 所属组g 其他人o 所有人a
+增加权限 -减少权限 =修改权限
权限数字表示
r — 4
w — 2
x — 1
rwxrw-r– 764

当在777文件夹下存在一个644文件夹普通用户可以删除644文件夹

文件
r：cat/more/head/tail/less 查看内容
w:vim 修改文件内容
x:script command 可执行文件

文件目录权限
| r | w | x |
| ——————– | ———————— | ———— |
| 读权限 | 写权限 | 执行权限 |
| 看文件内容 | 修改文件内容 | 执行文件内容 |
| 列出文件目录中的内容 | 创建删除文件或里面的目录 | 可以进入目录 |
| ls | touch/mkdir/rmidr/rm | cd |

更改权限：root和所属者

其他权限管理命令

权限管理命令：chown

命令英文原意: change file ownership
命令所在路径:/bin/chown
执行权限:所有用户
语法: chown[用户][文件或目录]
功能描述:改变文件或目录的所有者
useradd 添加用户

命令名称: chgrp

命令英文原意: change file group ownership
命令所在路径:/bin/chgrp
执行权限:所有用户
语法: chgrp[用户组][文件或目录]
功能描述:改变文件或目录的所属组
groupadd 添加组
缺省组就是默认组的意思

命令名称: umask

命令英文原意:the user file-creation mask
命令所在路径:Shell内置命令
执行权限:所有用户
语法: umask[-S]
-S以rwx形式显示新建文件缺省权限
功能描述:显示、设置文件的缺省权限
新创建的文件（touch）是没有可执行权限的（x）

umask 022
即为默认755
当mask的值是022—-w–w-
他的本来的权限与777是逻辑与的关系755目录，文件没有可执行权限即644
